# Expense Tracker

A simple web-based expense tracker using Firebase.

## Setup Instructions

1.  **Environment Configuration**:
    *   Copy `env.example.js` to `env.js`.
    *   Fill in your Firebase credentials in `env.js`.
    *   *Note: `env.js` is ignored by Git for security.*

2.  **Run Locally**:
    *   Open `index.html` in any modern web browser.

3.  **Deployment (GitHub Pages)**:
    *   To use this with GitHub Pages securely, you should use GitHub Actions to inject your credentials into `env.js` during the build process.
